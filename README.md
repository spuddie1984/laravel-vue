# Master Laravel 6 with Vue - FullStack Development
This is a project based off of the [udemy course](https://www.udemy.com/course/master-laravel-6-with-vuejs-fullstack-development) with the same name.

## Commits Overview

### 1st Commit - Laravel API & Vue routing Basics

- Install laravel and vue (using vagrant with homestead)
- setup .env with db info
- setup fallback route
- setup vue router
- add html5 history mode
- Register Vue components
- add bootstrap to vue 
- setup the basics of a basic navbar
- use v-bind (and its shorthand version) to bind named routes to router-link links
